### 两周自制脚本语言 示例代码

### virtual env 安装 ply依赖

```bash
$ python3 -m venv ply_env
$ source ply_env/bin/activate
$ pip install ply
```

### ubuntu apt 安装 ply依赖

```bash
$ sudo apt update
$ sudo apt install python3-ply  # Python 3
```
